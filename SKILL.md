# A 股股票深度分析 Skill

## 概述

本 Skill 通过 Tushare API 获取 A 股数据，从**基本面、技术面、资讯面**三个维度对个股进行深度分析，最终生成包含评分和投资建议的分析报告。

## 使用前提

- **Tushare Token**: 请通过环境变量 `TUSHARE_TOKEN` 配置
- **依赖**: curl, python3 (系统自带)

## 工作流程

### 直接运行分析脚本

```bash
# 分析指定股票
~/.openclaw/workspace/.agents/skills/a-stock-analyzer/scripts/analyze_stock.sh <股票代码> [输出目录]

# 示例
~/.openclaw/workspace/.agents/skills/a-stock-analyzer/scripts/analyze_stock.sh 600519
~/.openclaw/workspace/.agents/skills/a-stock-analyzer/scripts/analyze_stock.sh 002471 ~/.openclaw/workspace
```

### 输出结果

脚本会生成 JSON 格式的分析结果，包含：
- 基本面评分 (40%权重)
- 技术面评分 (30%权重)
- 资讯面评分 (30%权重)
- 综合评分和评级

## 分析框架

### 基本面分析（权重 40%）

| 子维度 | 核心指标 |
|--------|----------|
| 盈利能力 | ROE、EPS、净利率、毛利率 |
| 成长能力 | 净利润增速 |
| 估值水平 | 市净率、市盈率 |

### 技术面分析（权重 30%）

| 子维度 | 核心指标 |
|--------|----------|
| 趋势判断 | 60日涨跌幅 |
| 量价关系 | 成交量变化 |
| 波动性 | 量能对比 |

### 资讯面分析（权重 30%）

| 子维度 | 核心指标 |
|--------|----------|
| 概念热度 | 热门概念数量 |
| 资金流向 | 融资余额变化 |
| 股东结构 | 外资持股 |

## 综合评分

| 评分区间 | 评级 |
|----------|------|
| 80-100 | ★★★★★ 强烈推荐 |
| 60-79 | ★★★★ 值得关注 |
| 40-59 | ★★★ 中性 |
| 20-39 | ★★ 偏谨慎 |
| 0-19 | ★ 高风险 |

## 快速使用

在 OpenClaw 中直接说：

```
分析 600519
看看 002471 怎么样
帮我分析一下贵州茅台
```

## 输出示例

分析完成后会返回：
1. 综合评分和评级
2. 基本面/技术面/资讯面各单项评分
3. 核心数据摘要
4. 优势和风险提示

## 重要提示

本分析仅供参考，不构成投资建议。股市有风险，投资需谨慎。
