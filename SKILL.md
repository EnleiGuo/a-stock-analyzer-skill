# A 股股票深度分析 Skill

## 概述

本 Skill 通过 Tushare API 获取 A 股数据，从**基本面、技术面、资讯面**三个维度对个股进行深度分析，最终生成包含评分和投资建议的分析报告。

## 使用前提

- **Tushare Token**: 已配置在代码中 (TUSHARE_TOKEN_PLACEHOLDER)
- **依赖**: curl, python3 (系统自带)

## 工作流程

### 直接运行分析脚本

```bash
# 分析指定股票
~/.openclaw/workspace/.agents/skills/a-stock-analyzer/scripts/analyze_stock.sh <股票代码> [输出目录]

# 示例
~/.openclaw/workspace/.agents/skills/a-stock-analyzer/scripts/analyze_stock.sh 600519
~/.openclaw/workspace/.agents/skills/a-stock-analyzer/scripts/analyze_stock.sh 002471 ~/.openclaw/workspace
```

### 输出结果

脚本会生成 JSON 格式的分析结果，包含：
- 基本面评分 (40%权重)
- 技术面评分 (30%权重)
- 资讯面评分 (30%权重)
- 综合评分和评级

## 分析框架

### 基本面分析（权重 40%）

| 子维度 | 核心指标 |
|--------|----------|
| 盈利能力 | ROE、EPS、净利率、毛利率 |
| 成长能力 | 营收/净利润同比、3年CAGR |
| 估值水平 | PE/PB 5年历史分位数、PEG、股息率 |
| 偿债能力 | 流动比率、速动比率、资产负债率 |
| 现金流 | 每股经营现金流 vs EPS |
| 运营效率 | 总资产/存货/应收账款周转率 |

### 技术面分析（权重 35%）

| 子维度 | 核心指标 |
|--------|----------|
| 趋势判断 | MA均线系统、BOLL通道 |
| 量价关系 | 成交量异动、量价背离 |
| 技术指标 | MACD/RSI/KDJ背离检测（波峰波谷法） |

### 资金面分析（权重 25%）

| 子维度 | 核心指标 |
|--------|----------|
| 主力资金 | 特大单+大单净流入 |
| 融资融券 | 融资余额变化 |
| 股东增减持 | 近半年增减持统计 |
| 外资持仓 | 沪深港通持股变化 |

## 综合评分

| 评分区间 | 评级 |
|----------|------|
| 80-100 | ★★★★★ 强烈推荐 |
| 60-79 | ★★★★ 值得关注 |
| 40-59 | ★★★ 中性 |
| 20-39 | ★★ 偏谨慎 |
| 0-19 | ★ 高风险 |

## 快速使用

在 OpenClaw 中直接说：

```
分析 600519
看看 002471 怎么样
帮我分析一下贵州茅台
```

## 解读指引

生成报告后，在给出分析总结前，请先阅读 `references/analysis_framework.md` 获取 ROE 杜邦分解、利润含金量、估值分位数等方法论指导，以提升解读深度。

## 输出示例

分析完成后会返回：
1. 综合评分和评级（含三维雷达图）
2. 基本面/技术面/资金面各单项评分
3. K线蜡烛图、PE/PB历史走势图
4. 核心数据摘要与风险提示

## 重要提示

本分析仅供参考，不构成投资建议。股市有风险，投资需谨慎。
